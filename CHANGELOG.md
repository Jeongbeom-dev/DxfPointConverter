# 변경 이력 (CHANGELOG)

버전 형식: [주.부.수] — 주(호환 깨지는 변경) / 부(기능 추가) / 수(버그 수정)

## [1.1.0] - 2026-07-20
### 변경
- G-code 명령 접두를 `G01` → `G1` 로 변경
### 추가
- **한붓 최적화(점프 최소화)**: 외곽선+채우기를 그리디 최근접으로 재정렬/방향조정하여
  펜업(점프) 이동을 최소화 (열린선 역방향, 닫힌선 시작점 회전). 외곽선과 채우기를 함께 출력.
- **재생 강조 마커**: 재생 중 현재 그리는 지점을 링+십자 커서로 추적 표시
- 재생 속도 범위를 0.1~15 로 조정하고 **1 이하 저속**(소수 누적)을 지원

## [1.0.0] - 2026-07-20
최초 버전 관리 시작. 그동안 개발된 전체 기능 포함.

### 기능
- **DXF 파싱**: LINE, LWPOLYLINE(bulge 원호 포함), POLYLINE, CIRCLE, ARC, POINT
- **G-code 출력**: `G01<탭>X값<탭>Y값` (소수점 3자리), 경로 사이 빈 줄 = 펜업
- **헤더 입력**: 출력 상단에 삽입되는 헤더 편집창
- **저장**: `.gcode` 확장자
- **면채우기(슬라이스)**: 폐루프에 간격·방향(각도)·지그재그 스캔라인 해칭
- **기준점 오프셋**: 원본/중심/좌하/좌상/우하/우상
- **선분 결합(폐루프화)**: 끝점이 맞닿는 개별 선분을 이어 폐루프로 결합 (개별 LINE으로 그린 DXF 대응)
- **시각화**: 휠 줌·드래그 팬·화면맞춤·적응형 격자·원점 표시·펜업 점선·방향 화살표·그리기 순서 애니메이션·실시간 좌표
- **배포**: PyInstaller onefile — 타 PC에서 Python 없이 단독 실행

### 안정화
- 적대적 리뷰 워크플로 3회로 버그 22건 수정 (지그재그 해칭, scale clamp, bulge 정점 귀속,
  degenerate ARC, 재변환 시 상태 보존, 그리드 해시 경계, 조기 닫힘 등)
